# Escalera Inteligente

## Descripción
Este programa en Python simula una "escalera inteligente". Solicita el nombre del usuario y el piso actual, y luego imprime secuencialmente cada piso que la persona debe bajar hasta llegar al primer piso, garantizando una guía paso a paso.

## Funcionalidades
- Entrada de datos: Nombre y piso actual del usuario.
- Validación: Comprueba si el usuario ya está en el piso 1.
- Ciclo de descenso: Muestra el descenso piso por piso.
- Mensaje final: Notifica la llegada segura al primer piso.

## Cómo ejecutarlo
1. Asegúrate de tener instalado [Python]
2. Guarda el código en un archivo llamado `escalera.py`.
3. Ejecuta en tu terminal:
   ```bash
   python escalera.py


## Ejemplo de uso


```
--------------- ESCALERA INTELIGENTE ----------------
Ingrese el nombre del usuario: Juan
Ingrese el piso en el que se encuentra: 3
Estoy en piso 3. El siguiente paso es bajar uno.

Usuario Juan, baja al piso 2.
Estoy en piso 2. El siguiente paso es bajar uno.

Ha llegado al piso 1. Gracias por utilizar nuestros servicios.
```
